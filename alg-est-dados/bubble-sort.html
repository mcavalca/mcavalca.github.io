Bubble Sort


Definição


Considerado um dos mais simples, é um método para ordenação de dados, normalmente listas e vetores. O algoritmo irá percorrer os dados de entrada de modo sequencial - 1 até (n - 1) - e, em cada um, realizará uma comparação com o próximo elemento, realizando uma troca caso haja necessidade. Para uma ordenação em ordem crescente, a troca é realizada se o menor valor suceder o elemento atual, e vice-versa para uma ordenação decrescente. A complexidade deste algoritmo é de ordem quadrática, ou seja, no pior e médio caso possui uma complexidade de O(n2).

Códico em C



        void BubbleSort(int *vetor, int tamanho){
            int i, j, troca;
            for (i = 0; i < tamanho; i++){
                for (j = 0; j < (tamanho - 1); j++){
                    if (*vetor[j] > *vetor[j + 1]){
                        troca = *vetor[j];
                        *vetor[j] = *vetor[j + 1];
                        *vetor[j + 1] = troca;
                    }
                }
            }
        }

Este algoritmo é extremamente simples, e não checa se o vetor já está ordenado antes de realizar as trocas. Para isso basta adicionar uma condição que irá verificar, após cada iteração, se o vetor já está ordenado ou não.


Bubble Sort "Inteligente"


Este algoritmo possui a mesma essência do anterior, porém com uma condição que irá verificar se o vetor já está ordenado após cada iteração. Segue abaixo o algoritmo em C com esta nova função.

Códico em C



        void BubbleSortSmart(int *vetor, int tamanho){
            int i, j, troca;
            for (i = 0; i < tamanho; i++){
            
                // Esta variável irá dizer se houve troca ou não...
                int houveTroca = 0; 
                // A troca só é realizada se o vetor não estiver ordenado
                
                for (j = 0; j < (tamanho - i); j++){
                    if (*vetor[j] > *vetor[j + 1]){
                        troca = *vetor[j];
                        *vetor[j] = *vetor[j + 1];
                        *vetor[j + 1] = troca;
                        houveTroca = 1;
                    }
                }
                
                // ... caso não houve troca, é porque o vetor já está ordenado
                if (!houveTroca) break;
            }
        }
