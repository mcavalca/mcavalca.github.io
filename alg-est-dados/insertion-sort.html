Insertion Sort


Definição


Este algoritmo de ordenação utiliza do mesmo princípio do Selection Sort: através de uma lista de entrada utiliza um marcador que parte do início do vetor para dividí-lo e insere o elemento marcado na sua posição correta do sub-vetor à esquerda, ajustando a posição de todos os outros elementos, e então avança o marcador, até que tenha percorrido todo o vetor. Do mesmo modo que o Selection Sort funciona, este algoritmo de ordenação irá gerar 2 sub-vetores à partir do marcador: o sub-vetor à esquerda contêm a parte já ordenada temporariamente, e à direita a parte não ordenada.

Podemos comparar este método de ordenação ao modo como podemos ordenar um conjunto de cartas: digamos que o jogador já tenha um grupo de cartas já ordenado em sua mão. Ao receber uma nova carta, o jogador irá movê-la para a posição correta deste grupo e, assim, terá o conjunto ordenado novamente.

Este método de ordenação possui complexidade de:

Melhor Caso: O(n)
Caso Médio: O(n2)
Pior Caso: O(n2)

        void InsertionSort(int *vetor, int tamanho) {

            int troca;

                j = (i - 1);
                while (troca < v[j] && j >= 0) {
                    v[j + 1] = v[j];
                    j--;
                }
                v[j + 1] = troca;
            }
            
        }
