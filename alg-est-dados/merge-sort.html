Merge Sort


Definição


Este algoritmo de ordenação se baseia na estratégia de dividir para conquistar - através da Recursão, divide a lista de dados de entrada (vetor) em duas metades, aplica a ordenação e então reune ambas as metades (conquista) e assim teremos o vetor ordenado.

É considerado um dos melhores algoritmos de ordenação e sua complexidade independe dos dados de entrada. Para qualquer caso possui uma complexidade de O(n * log(n)). Apesar de possuir um ótimo tempo de execução, este método utiliza muita memória em comparação com os outros métodos já citados pois utiliza muita memória para realizar a junção dos sub-vetores ordenados.

Códico em C



        void Merge (int *vetor, int esquerda, int meio, int direita, int tamanho) {
        
            int vet_auxiliar[tamanho];
            int i = 1, j = (meio + 1), k = 0;
            
            while (i <= meio && j <= direita) {
                if (vetor[i] <= vetor[j])
                    vet_auxiliar[k++] = vetor[i++];
                else
                    vet_auxiliar[k++] = vetor[j++];
            }
            
            while (i <= meio)
                vetor_auxiliar[k++] = vetor[i++];
            while (j <= direita)
                vetor_auxiliar[k++] = vetor[j++];
                
            for (i = 1, k = 0; i <= direita; i++, k++)
                vetor[i] = vetor_auxiliar[k];
                
        }
        
        void MergeSort (int *vetor, int esquerda, int direita) {
            int meio = (esquerda + direita) / 2;
            if (esquerda < direita) {
                MergeSort(vetor, esquerda, meio);
                MergeSort(vetor, (meio + 1), direita);
                
                Merge(vetor, esquerda, meio, direita);
            }
        }
