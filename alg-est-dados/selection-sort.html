Selection Sort


Definição


A idéia deste algoritmo de ordenação é selecionar o menor elemento da lista de entrada (vetor) e o trocar com o elemento que está no início do vetor. Isto é feito com a ajuda de um marcador que irá dividir o vetor em 2 partes - ordenada, à sua esquerda, e desordenada, à sua direita. O algoritmo então irá procurar pelo menor elemento no sub-vetor desordenado e trocar com o elemento no início deste mesmo sub-vetor. Após a verificação no último elemento deste sub-vetor, o marcador irá avançar para o próximo elemento. Este processo se repete até que o marcador tenha percorrido todo o vetor. Ao longo da execução afirma-se que os elementos do sub-vetor à esquerda já estão ordenados de forma correta.
A complexidade deste algoritmo é de O(n2) para todos os casos.

Códico em C



        void SelectionSort(int *vetor, int tamanho) {
        
            int i, j, iMenor, troca;
            for (i = 0; i < (n - 1); i++) {
                iMenor = i;
                for (j = (i + 1); j < n; j++) {
                    if (vetor[j] < vetor[iMenor]) {
                        iMenor = j;
                    }
                }
            }
            
            troca = vetor[iMenor];
            vetor[iMenor] = vetor[i];
            vetor[i] = troca;
        }
